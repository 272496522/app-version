## APP版本管理系统

APP版本管理是一套多应用更新发版的管理平台。用户可以通过部署本管理系统，以实现对多APP的多平台多渠道上的版本管理。

### 项目结构

```
app-version
│   README.md
└───modules
│   └───app-version-common (公共代码 实体类 工具类等)
│   └───app-version-manager (后台管理模块)
│   └───app-version-rest (与APP交互的Rest接口模块)
└───web
    └───dashboard (前端)
```
